MI PLAN FITNESS — INSTALACIÓN

Este paquete es una PWA. Para que funcione como app en iPhone debe estar publicado en internet mediante HTTPS.

Archivos:
- index.html
- plan.js
- manifest.webmanifest
- sw.js

Opción sencilla:
1. Crea una cuenta gratuita en GitHub.
2. Crea un repositorio nuevo.
3. Sube estos cuatro archivos.
4. Activa GitHub Pages desde Settings > Pages.
5. Abre el enlace publicado en Safari.
6. Compartir > Agregar a pantalla de inicio.

La dieta, rutina y lista del súper pueden editarse dentro de la propia app en la pestaña “Editar plan”.
No contiene información sobre fármacos.
